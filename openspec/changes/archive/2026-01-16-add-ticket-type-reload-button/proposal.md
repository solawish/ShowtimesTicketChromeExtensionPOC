# Change: 在票種下拉選單旁新增重新讀取按鈕

## Why
當票種資料需要更新時（例如：票種價格變更、新增票種選項等），使用者目前只能透過重新選擇時間來觸發票種資料的重新載入。提供一個明確的「重新讀取」按鈕可以讓使用者更方便地手動重新載入票種資料，無需重新選擇時間。

## What Changes
- 在票種下拉選單旁邊新增一個「重新讀取」按鈕
- 按下按鈕後重新呼叫票種 API 並更新票種選單選項
- 按鈕應在票種選單啟用時才可點擊（即已選擇時間後）
- 重新載入時應顯示載入狀態提示
- 重新載入時應保留當前選定的票種（如果該票種仍存在於新的資料中）

## Impact
- Affected specs: `ticket-configuration`
- Affected code: `popup.html`, `popup.js`, `popup.css`
